# Rapido-Analysis-SQL
‚úÖ Project Overview

The Rapido Ride Analytics project is focused on analyzing the operational, customer, driver, and revenue performance of the Rapido bike taxi platform.
The dataset contains 3,000+ rides, 500 users, and 300 drivers, linked across three tables:

rapido1 (User Profiles)

rapido_drivers2 (Driver Profiles)

rapido_rides_with_names3 (Ride Transactions)

The goal is to convert raw ride logs into meaningful insights that help improve service efficiency, pricing decisions, driver management, and customer satisfaction.
